Ceylon Coffee Website

A modern and creative coffee shop website built using HTML, CSS, and JavaScript.  
Designed with a premium dark aesthetic, smooth animations, responsive layout, and interactive UI components.

Features

  Modern responsive design
  Mobile-friendly navigation
  Smooth scrolling effects
  Interactive hover animations
  Stylish coffee menu section
  Customer testimonials
  Creative hero section
  Responsive cards and layouts
  Clean and organized code structure

Technologies Used

  HTML5
  CSS3
  JavaScript (Vanilla JS)
  Google Fonts

Responsive Design

The website is fully responsive and works on:

  Desktop
  Laptop
  Tablet
  Mobile devices
